.qr.util.tbl.crossTab

crosstab statics on table

Parameters:
  • b: - group by columns
  • p: - pivot column, column type must be symbol
  • t: - Q table
  • v: - exposed value v
Return: the crosstab *pivot table

.qr.util.tbl.isKeyedTbl

check if input is a keyed tables

Parameters:
  • tbl: - input data. Can be anything
return:: return true if input data is keyed table. False otherwise

.qr.util.tbl.isNonEmptyTbl

check if input is non-empty tables

Parameters:
  • tbl: - input data. Can be anything
return:: return true if input data is not an empty table. False otherwise

.qr.util.tbl.lj

Safely left join table on columns specified

Parameters:
  • colsToJoin: - columns to join both table
  • lTabl: - tables on the left for left join
  • rTbl: - tables on the right for left join
return:: a table contains resulted from left joining the lTbl with rTbl

.qr.util.tbl.prepends

prepend the column names with prefix

Parameters:
  • exceptCols: - exceptional columns which will NOT be appended
  • prefix: - prefix to be appended in front of the column name
  • tbl: - a table
return:: the table with column names appended by prefix

.qr.util.tbl.splitCol

split the string column and ungroup

Parameters:
  • c: - column to ungroup, in symbol
  • t: - Q table
Return: the columns ungroup by c

.qr.util.tbl.uj

Safely union join table on columns specified

Parameters:
  • lTabl: - tables on the left for left join
  • rTbl: - tables on the right for left join
return:: a table contains resulted from union joining between two tables
