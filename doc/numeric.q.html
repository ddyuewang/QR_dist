.qr.quant.numeric.bisection

find the root using bisection method

Parameters:

  • b: - right point on x

Return: the root

.qr.quant.numeric.integral

compute the integral of f using Adaptive Simpson's Method

Parameters:
  • a: - start point
  • b: - end point
  • func: - integrated function
Return: the numerical integral of func integrated from a to b
