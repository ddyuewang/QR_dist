Entity Details

.qr.util.schema.addCol

construct the schema entry given the colName and type

Parameters:
  • colNameIn: - the name of column
  • typeNameIn: - must be Q datatype, with suffix "s" to indicate as list
Return: the schema entry

.qr.util.schema.addTbl

register the schema with table name

param::
  • schemaIn, - schema to register, constructed by .qr.util.schema.addCol
  • tablNameIn, - the name of the schema

.qr.util.schema.formatTbl

format the table given registered schema

Parameters:
  • tbl: - a Q table
  • tblNameIn: - the name of the schema
Return: the table formated by Q schema, with column types automatially promoted

.qr.util.schema.getEmptyTbl

return the empty table given registered schema

Parameters:
  • tblNameIn: - the name of the schema
Return: the corresponding empty table with meta

.qr.util.schema.getSchemaCodes

return the codes to construct the corresponding schema given table

Parameters:
  • tbl: - a Q table
Return: the schema codes matching the meta of tbl
