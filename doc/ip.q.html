.qr.quant.bip.solve

solve min(max) y'b, s.t. y'A >=(<=) c, with y as binary (1 or 0) using relaxation to lp with simplex algorithm

Parameters:
  • A: - constraint matrix
  • b: - objective function coefficients
  • constr: - constraint vector
Return: the result in a dictionary with optimal solution
dir:: objective direction, either `max or `min
method:: indicate which method to use, must be one either `explicit or `implicit
